# Menu Bar Icon — Fringer Status Item

## 用途
显示在 macOS 菜单栏（右上角）的小图标，用户点击它来切换 Fringer Bar。
当前使用 SF Symbol `line.3.horizontal.decrease`，需要替换为自定义图标以增强品牌辨识度。

## 技术要求
- **尺寸**: 18x18 pt（需要 @1x 和 @2x，即 18px 和 36px）
- **格式**: PNG，透明背景
- **颜色**: 纯黑/纯白（必须设为 template image，macOS 会根据亮/暗模式自动着色）
- **风格**: 极简线条图标，笔画粗细约 1.5-2pt，与系统 SF Symbols 风格协调

# Empty State Illustration — Triggers 空状态

## 用途
触发器（Triggers）页面没有任何触发器时显示的空状态插图，替代当前的 SF Symbol `bolt.circle`。

## 技术要求
- **尺寸**: 96x96 像素（@2x 即 192x192）
- **格式**: PNG，透明背景

# Menu Bar Icon (Active State) — Fringer Bar Visible

## 用途
当 Fringer Bar（溢出面板）展开时，菜单栏图标切换为"激活"状态，让用户知道隐藏项目正在显示。

## 技术要求
- 与 02_MenuBarIcon 相同尺寸和格式（18x18 pt, @1x/@2x）
- PNG，透明背景，纯黑 template image
- 应与普通状态图标明显不同但风格一致

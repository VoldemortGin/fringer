# Empty State Illustration — Presets 空状态

## 用途
预设（Presets）页面没有任何预设时显示的空状态插图，替代当前的 SF Symbol `rectangle.stack`。

## 技术要求
- **尺寸**: 96x96 像素（@2x 即 192x192）
- **格式**: PNG，透明背景
